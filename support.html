常見問題

如何連線到我的 Mac？

1. 在 Mac 上開啟「系統設定」→「一般」→「共享」→ 啟用「螢幕共享」。

2. 記下 Mac 的 IP 位址（可在「系統設定」→「網路」中查看）。

3. 在 VNC Pad 中輸入 IP 位址和 VNC 密碼即可連線。

如何啟用注音/拼音/日文輸入法支援？

您需要在 Mac 上安裝 VNCRemoteHelper。請參閱 Helper 安裝教學。

連線後畫面很模糊怎麼辦？

VNC Pad 會根據網路狀況自動調整畫質。如果網路穩定但畫面仍然模糊，請嘗試在設定中手動選擇「高品質」模式。

外接鍵盤的快捷鍵沒有反應？

請確認已安裝 VNCRemoteHelper 並已授予「輔助使用」權限。詳見 Helper 安裝教學。

可以透過網際網路連線嗎？

VNC Pad 設計為區域網路使用。如需透過網際網路連線，建議使用 VPN 將兩端設備置於同一虛擬網路中。

系統需求

  • iPadOS 16.0 或以上版本
  • 目標 Mac 需啟用「螢幕共享」
  • 兩台裝置需在同一區域網路
  • 建議使用外接鍵盤以獲得最佳體驗

Frequently Asked Questions

How do I connect to my Mac?

1. On your Mac, go to System Settings → General → Sharing → enable Screen Sharing.

2. Note your Mac's IP address (found in System Settings → Network).

3. Enter the IP address and VNC password in VNC Pad to connect.

How do I enable CJK input method support?

You need to install VNCRemoteHelper on your Mac. See the Helper Setup Guide.

The screen looks blurry after connecting?

VNC Pad automatically adjusts quality based on network conditions. If your network is stable but the image is still blurry, try manually selecting "High Quality" mode in settings.

External keyboard shortcuts not working?

Make sure VNCRemoteHelper is installed and has Accessibility permissions enabled. See the Helper Setup Guide.

Can I connect over the internet?

VNC Pad is designed for local network use. For internet access, we recommend using a VPN to place both devices on the same virtual network.

System Requirements

  • iPadOS 16.0 or later
  • Screen Sharing enabled on the target Mac
  • Both devices on the same local network
  • External keyboard recommended for best experience
