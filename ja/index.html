よくある質問

なぜ他のリモートデスクトップアプリでは日本語入力ができないのですか？

従来の VNC プロトコルは X11 keysym コードしか送信できず、macOS の入力メソッドの変換プロセスをトリガーできません。VNC Pad は macOS 側で Helper アプリを実行し、HID usage code を独立した TCP チャネル（ポート 50001）で送信、CGEvent HID tap レベルでインジェクションします。macOS はこれらを物理キーボード入力として扱い、入力メソッドが正常に処理されます。

Helper アプリなしで使えますか？

はい。Helper なしでも VNC Pad のすべての機能は正常に動作します。ただし、日本語、中国語、韓国語などの非英語入力メソッドは使用できません。英語入力とすべてのキーボードショートカットは影響を受けません。

インターネット経由で接続できますか？

VNC Pad はローカルネットワーク用に設計されています。インターネット経由でアクセスするには、VPN を使用して両方のデバイスを同じ仮想ネットワークに配置することをお勧めします。

Helper セットアップガイド

macOS Helper をインストールして、日本語入力などを有効にします。
