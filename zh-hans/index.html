常见问题

为什么其他远程桌面 App 不能打拼音？

传统 VNC 协议只能传送 X11 keysym 键码，这些键码只代表单一字符，无法触发 macOS 输入法的组字流程。VNC Pad 通过在 macOS 端运行 Helper 辅助程序，将 HID usage code 通过独立 TCP 通道（port 50001）传送并在 CGEvent HID tap 层级注入，macOS 会将这些事件视为实体键盘输入，因此输入法可以正常处理。

不安装 Helper 可以用吗？

可以。不安装 Helper 的话，VNC Pad 所有功能都正常运行，只是无法使用拼音、注音、日文等非英文输入法。英文输入和所有快捷键不受影响。
