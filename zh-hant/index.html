常見問題

為什麼其他遠端桌面 App 不能打注音？

傳統 VNC 協定只能傳送 X11 keysym 鍵碼，這些鍵碼只代表單一字元，無法觸發 macOS 輸入法的組字流程（如注音的聲母韻母序列）。VNC Pad 透過在 macOS 端執行 Helper 輔助程式，將 HID usage code 透過獨立 TCP 通道（port 50001）傳送並在 CGEvent HID tap 層級注入，macOS 會將這些事件視為實體鍵盤輸入，因此輸入法可以正常處理。

不安裝 Helper 可以用嗎？

可以。不安裝 Helper 的話，VNC Pad 所有功能都正常運作，只是無法使用注音、拼音、日文等非英文輸入法。英文輸入和所有快捷鍵不受影響。

可以透過網際網路連線嗎？

VNC Pad 設計為區域網路使用。如需透過網際網路連線，建議使用 VPN 將兩端設備置於同一虛擬網路中。
