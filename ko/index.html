macOS 입력기를 완벽 지원하는 유일한 iPad 원격 데스크톱

VNC Pad는 $9.99의 iPad 원격 데스크톱 앱(VNC 클라이언트)으로, macOS 화면 공유에 연결합니다. 한국어, 중국어(주음/병음), 일본어 입력기를 지원하는 유일한 iPad VNC 클라이언트입니다. HID 인젝션 기술로 구현. Metal GPU 가속 렌더링, 232+ 키 매핑, 적응형 화질 제어를 탑재하며, 완전히 로컬 네트워크에서 작동하고 클라우드 계정이 필요 없습니다.

왜 VNC Pad인가?

⌨

완벽한 입력기 지원

원격 Mac에서 한국어, 중국어(주음/병음), 일본어 입력기가 완벽하게 작동하는 시장 유일의 iPad 앱. HID 인젝션 기술로 macOS가 iPad 외장 키보드를 네이티브 물리 키보드로 인식 — 모든 입력기가 그대로 작동하며, 조합과 후보 선택이 완벽하게 동작합니다.

✗ 다른 원격 데스크톱 앱: VNC 프로토콜은 영문 키코드만 전송 가능, 한글 입력기가 제대로 작동하지 않음

✓ VNC Pad: HID 레벨 인젝션으로 macOS 네이티브 입력기 완벽 작동

🖥

Metal 하드웨어 가속 렌더링

Apple Metal API로 GPU 직접 렌더링. Dirty Region 부분 업데이트와 제로카피 텍스처 업로드로 최저 지연시간과 화면 티어링 제로를 실현합니다.

🎹

232+ 키 완전 매핑

좌우 분리된 9개 수정자 키를 개별 추적. iPadOS 시스템 단축키(Cmd+Tab, Ctrl+Space 등)를 인터셉트하여 모든 키 입력을 원격 Mac에 확실히 전달합니다.

📡

스마트 적응형 화질 제어

네트워크 지연시간을 실시간 모니터링하여 고화질/균형/저대역폭 모드를 자동 전환. 네트워크 변동 시 자동 조정되어 수동 설정이 필요 없습니다.

🖱

120Hz 트랙패드 추적

iPad 외장 트랙패드와 마우스를 네이티브 지원. 로컬 커서 예측으로 체감 지연시간을 줄이고, 스마트 스크롤 가속도 보정으로 부드러운 조작감을 제공합니다.

🔒

프라이버시 최우선

완전 로컬 네트워크 동작. 계정 등록 불필요, 클라우드 서버를 일절 거치지 않습니다. 연결 데이터는 모두 기기에 저장, 비밀번호는 iOS Keychain으로 안전하게 관리합니다.

자주 묻는 질문

왜 다른 원격 데스크톱 앱에서는 한국어 입력이 안 되나요?

기존 VNC 프로토콜은 X11 keysym 코드만 전송할 수 있어 macOS 입력기의 조합 프로세스를 트리거할 수 없습니다. VNC Pad는 macOS에서 Helper 앱을 실행하고, HID usage code를 독립 TCP 채널(포트 50001)로 전송하여 CGEvent HID tap 레벨에서 인젝션합니다. macOS는 이를 물리 키보드 입력으로 처리하여 입력기가 정상 작동합니다.

Helper 앱 없이 사용할 수 있나요?

네. Helper 없이도 VNC Pad의 모든 기능이 정상 작동합니다. 다만 한국어, 중국어, 일본어 등 비영어 입력기를 사용할 수 없습니다. 영어 입력과 모든 키보드 단축키는 영향받지 않습니다.

인터넷을 통해 연결할 수 있나요?

VNC Pad는 로컬 네트워크용으로 설계되었습니다. 인터넷을 통해 접속하려면 VPN을 사용하여 두 기기를 같은 가상 네트워크에 배치하는 것을 권장합니다.
